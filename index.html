git status

O git status verifica a situação do seu repositório

git add

Prepara as alterações realizadas no seu repositório local

git commit

O git commit confirma as suas alterções no seu repositório local

git push

Envia as alterações da pasta local ao repositório remoto

git pull

Baixa as alterações do seu repositório remoto
